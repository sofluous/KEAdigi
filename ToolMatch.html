How to Play

Each round tells you a job from a drawing or design app. Tap the tool that matches what you would use.

  • Look at the icon first, then think about what that tool usually does.
  • Some tools change color, some add objects, and some edit what is already there.
  • Build a streak by matching several jobs in a row.
