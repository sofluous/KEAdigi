How to Play

This module explains motion by showing a short sequence of still frames.

  • Choose a source frame set like 6, 12, 24, 30, or 60 frames.
  • Use Play to let that frame set run automatically.
  • Change playback speed to compare the same frames at different rates.
  • Scroll the timeline when a large frame set does not fit all at once.
