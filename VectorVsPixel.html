How to Play

This module compares two ways a computer can store the same picture.

  • Choose a sample drawing.
  • Zoom in to compare the smooth vector edge with the pixel grid.
  • Change the raster grid size to see how resolution changes the pixel version.
  • Notice that the vector drawing stays smooth while the pixel view shows blocks.
